Netcdf4安装教程

发表于 2017-05-19 | 分类于 software | | 阅读次数

本指南的目的在于实现netcdf4在个人PC或是服务器上的安装，主要实现功能是netcdf4中的数据压缩（Compression）功能，若需要其他功能，请参阅官方指南：Getting and Building netCDF。

所需安装包

实现Data Compression所必须的安装包只有四个

  • HDF5 1.8.9 or later (for netCDF-4 support)
  • zlib 1.2.5 or later (for netCDF-4 compression)
  • netcdfc (latest version)
  • netcdf-fortran (latest version)

另：非必装：curl 7.18.0 or later (for DAP remote access client support)

注意：

  • 编译器的选择，在 /.configure中需要进行相关配置，也可在其之前就进行配置。在本次安装过程中，所需的C编译器和fortran编译器分别为intel的icc和ifort。
  • 如果不需要压缩功能，可以直接忽略1,2步骤
